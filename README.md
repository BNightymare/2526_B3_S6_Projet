# 2526_B3_S6_Projet

Project for B3 S6 - Hardware and Software Integration

## Project Structure

```
2526_B3_S6_Projet/
├── hardware/
│   └── mechanical/       # Mechanical design files
│       ├── cad/          # CAD models (STEP, STL, etc.)
│       ├── assembly/     # Assembly instructions
│       ├── drawings/     # Technical drawings
│       ├── specifications/ # Mechanical specifications
│       ├── bom/          # Bill of Materials
│       └── README.md     # Mechanical documentation
├── docs/                 # Project documentation
│   └── MECHANICAL_INTEGRATION_GUIDELINES.md
└── README.md            # This file
```

## Components

### Mechanical Integration
The `hardware/mechanical/` directory contains all mechanical design files, CAD models, assembly instructions, and technical drawings. See [hardware/mechanical/README.md](hardware/mechanical/README.md) for details.

## Getting Started

### Mechanical Design
1. Navigate to `hardware/mechanical/` directory
2. Read the README and mechanical integration guidelines
3. Choose your CAD tool (Fusion 360, SolidWorks, FreeCAD, etc.)
4. Create 3D models and save in `cad/` directory
5. Generate technical drawings in `drawings/` directory
6. Document assembly process in `assembly/` directory
7. Maintain BOM in `bom/` directory
