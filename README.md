# 2526_B3_S6_Projet

Project for B3 S6 - Hardware and Software Integration

## Project Structure

```
2526_B3_S6_Projet/
├── hardware/
│   └── pcb/              # PCB design files
│       ├── schematic/    # Circuit schematics
│       ├── layout/       # PCB layouts
│       ├── gerbers/      # Manufacturing files
│       ├── bom/          # Bill of Materials
│       └── README.md     # PCB documentation
├── docs/                 # Project documentation
│   └── PCB_DESIGN_GUIDELINES.md
└── README.md            # This file
```

## Components

### PCB (Printed Circuit Board)
The `hardware/pcb/` directory contains all PCB design files and documentation. See [hardware/pcb/README.md](hardware/pcb/README.md) for details.

## Getting Started

### PCB Design
1. Navigate to `hardware/pcb/` directory
2. Read the README and design guidelines
3. Choose your PCB design tool (KiCad, Altium, Eagle, etc.)
4. Start with schematic design
5. Create PCB layout
6. Generate manufacturing files (Gerbers)
