# Schematic Directory

Place your circuit schematic files here.

## Supported Formats
- KiCad (.sch, .kicad_sch)
- Eagle (.sch)
- Altium (.SchDoc)
- EasyEDA formats

## Naming Convention
- Use descriptive names: `projectname_schematic_v1.0.sch`
- Include version numbers
- Keep separate sheets for different subsystems
