# Layout Directory

Place your PCB layout files here.

## Supported Formats
- KiCad (.kicad_pcb)
- Eagle (.brd)
- Altium (.PcbDoc)
- Gerber files (RS-274X)

## Best Practices
- Keep layout files synchronized with schematics
- Use version control for all changes
- Export Gerber files after major changes
- Document layer stackup if multi-layer

## Layer Naming
- Top Layer (Copper)
- Bottom Layer (Copper)
- Silkscreen Top
- Silkscreen Bottom
- Solder Mask Top
- Solder Mask Bottom
- Paste Mask (if required)
